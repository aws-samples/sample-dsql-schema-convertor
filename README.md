# Aurora DSQL Schema Converter

> **Live tool:** https://noql.github.io/dsql-schema-convertor/

A web tool that converts PostgreSQL DDL schemas to Aurora DSQL-compatible format using deterministic rule-based transformations — runs entirely in the browser with no backend required.

## What it converts

| PostgreSQL Feature | DSQL Conversion |
|---|---|
| SERIAL / BIGSERIAL / SMALLSERIAL | Equivalent integer type + `GENERATED BY DEFAULT AS IDENTITY (CACHE 1)` |
| IDENTITY columns | Kept, ensures CACHE is specified (1 or >= 65536) |
| Sequences | Kept, adds explicit `CACHE 1` if missing, fixes invalid CACHE values |
| Foreign Keys | Removed (enforce in application) |
| Triggers | Removed (move to application) |
| PL/pgSQL Functions | Removed (move to application or rewrite as SQL) |
| SQL Functions | Kept (LANGUAGE SQL is supported) |
| Extensions | Removed |
| Table Inheritance (INHERITS) | Removed |
| Partitioning | Removed (DSQL auto-distributes) |
| Rules | Removed |
| LISTEN/NOTIFY | Removed |

## Deployment

Push to `main` — GitHub Actions builds and deploys automatically.

## Local Development

```bash
npm install
npm start
```

## Tech Stack

- **Frontend:** React + [AWS Cloudscape Design System](https://cloudscape.design/)
- **Conversion:** Deterministic regex-based rules in the browser (`src/converter.js`)
- **Hosting:** GitHub Pages (built via GitHub Actions)
